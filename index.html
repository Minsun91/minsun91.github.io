CLIPS

Application for managing appointments and recommending places around users

The project is a service that assists friends in deciding where to meet, recommending random restaurants, bars, cafes, etc., located at the midpoint between users' locations. As a backend developer on this project, I performed the following tasks:

  • Recommend place by using map/local API, depending on user location.
  • Crawling images and business hours of recommended places.
  • Using Sequelize to create a database and establish each table relationship.
  • Using multer-S3 to create/read/update/delete multiple review images.

  • Tech Stack: Node.js, Sequelize, AWS EC2, AWS S3, puppeteer, axios, cheerio
  • Github : https://github.com/codeing999/CLIPs-backend
  • Period : 08.2022-10.2022
